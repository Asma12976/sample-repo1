# Contact Manager Web App

This is a simple interactive contact management web application built with HTML, CSS, Bootstrap, and JavaScript.

## Features
- Add new contacts
- Edit existing contacts
- Delete contacts
- List and search contacts interactively
- Responsive design using Bootstrap
- Data stored in browser localStorage

## How to Use
1. Open `index.html` in your web browser.
2. Use the form to add, edit, or delete contacts.
3. Search contacts using the search bar.

## Technologies Used
- HTML
- CSS
- Bootstrap 5
- JavaScript

## File Structure
- `index.html` – Main HTML file
- `styles.css` – Custom styles
- `script.js` – App logic
